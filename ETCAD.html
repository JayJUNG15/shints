ETCAD 개선 요청 리스트 (베트남 공장 피드백 번역)

1. 전체 개요

베트남 공장에서 ETCAD 사용 중 발생한 기능 개선 요청 사항을 정리한 문서입니다.
각 항목별로 현재 개선 상태 및 추가 개선 필요 사항을 포함하고 있습니다.

2. 상세 항목

1) 설계 상 대칭 뒤집기 기능

  • 상태: 개선 완료
  • 추가 개선: 없음 (OK)

2) Develop Point 기능

  • 상태: 개선 완료
  • 추가 개선: 없음 (OK)

3) 패턴 분리 시 색상 구분 (사이즈별)

  • 요청 내용:
    "연결(link)" 상태의 디테일 색상이 사이즈별로 다르게 표시되도록
  • 상태: 개선 완료 (2026-04-10)
  • 추가 개선: 없음 (OK)

4) 멀티 컬러 원단 적용

  • 요청 내용:
    한 의류 내 여러 색상 원단 조합 가능하도록
  • 상태: 개선 완료 (2026-04-10)
  • 추가 개선: 없음 (OK)

5) 직선 분할 시 그레이딩 유지

  • 요청 내용:
    디자인에서 직선을 여러 구간으로 나누더라도
    분리된 패턴의 그레이딩(step)이 깨지지 않도록
  • 상태: 개선 완료 (2026-04-10)
  • 추가 개선: 없음 (OK)

6) 곡선 구간 시접 처리 (3cm 이상)

  • 요청 내용:
    곡선 구간에서 3cm 이상의 시접(seam allowance) 생성
  • 요청일: 2026-02-23
  • 상태: 미개선
  • 추가 개선: ET CAD 측 대응 대기
  • 참고: 이미지 1

7) 패턴 연결(Link) 기능

  • 상태: 개선되었으나 여전히 부족

추가 개선 필요 사항:

  1. 병합된 직선에서 그레이딩(step)이 깨지지 않도록 개선 필요
  2. 디자인에서 내부선을 삭제하면, 분리된 패턴에서도 동일하게 삭제되어야 함
  3. 디자인에서 외곽선을 삭제하면, 분리된 패턴에서도 동일하게 삭제되어야 함
  4. 외곽선 삭제 시 분리 패턴에서도 동일하게 반영 필요 (중복 요청)
  5. 디자인 수정 시:
    • 분리 패턴은 링크 상태(파란색)로 유지되나
    • 실제 내용은 원본과 불일치 (Sync 문제 발생)
  • 참고: 영상 1

8) 마커 구성 방식 (사이즈별 길이 계산 문제)

  • 상태: 미개선

문제 내용:

  • Lectra

    • 사이즈별 개별 마커 생성 → 이후 하나로 결합
    • 각 사이즈별 길이 계산 가능

  • ETCAD

    • 처음부터 모든 사이즈를 하나의 마커로 생성
    • 사이즈별 길이 확인 불가

영향:

  • 각 사이즈별 원단 소요량(Consumption) 계산 어려움

  • 참고: 영상 2 + 이미지 2

9) 패턴 수정 시 마커 재작업 문제

  • 상태: 미개선

문제 내용:

  • Lectra

    • 패턴 변경 시 1번의 명령으로 마커 자동 정리
    • 겹침 없음

  • ETCAD

    • 패턴 변경 시 마커를 처음부터 다시 생성해야 함
    • 작업 시간 과다 소요

  • 참고: 영상 3

10) Lectra Grading 기능 요청

  • 상태: 미구현

요청 내용:

  • Lectra에는 grading(사이즈 확장)에 매우 유용한 2가지 기능 존재
  • 관련 영상 공유 완료

요청 사항:

  • ETCAD에도 동일 또는 유사 기능 개발 요청

  • 참고: 영상 4 + 5

3. 요약

  • 완료 항목: 1 ~ 5
  • 부분 개선: 7
  • 미개선 핵심 항목: 6, 8, 9, 10

4. 핵심 이슈

  • 패턴 Link Sync 불안정
  • 마커 구조 문제 (사이즈별 분석 불가)
  • 패턴 변경 시 마커 재작업 필요
  • 일부 핵심 기능 (Lectra 대비) 미구현
